What I do

I specialize in software testing and quality assurance, combining automation expertise and manual testing precision to deliver reliable software solutions. Here are some key areas where I excel:

Test Automation

Developing robust and maintainable test automation frameworks using tools like Selenium, Cypress, and Appium. My focus is on improving efficiency and reducing manual effort.

Manual Testing

Performing in-depth manual testing to ensure flawless user experiences by identifying potential issues and addressing them proactively.

API Testing

Validating APIs using tools like Postman and RestAssured to ensure seamless communication between systems and robust back-end performance.

Performance Testing

Conducting performance and load testing with JMeter to ensure software can handle high traffic and deliver reliable results under stress.

Collaboration

Working closely with developers, product owners, and QA teams to ensure efficient workflows, detailed documentation, and shared goals.

Continuous Learning

Staying up-to-date with the latest industry trends, testing tools, and methodologies to deliver innovative and effective solutions.
